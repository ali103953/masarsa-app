# Executive English KSA

WebView app for:
https://masarsa.online/

## App Info
App Name: Executive English KSA
Bundle ID suggestion: com.masarsa.executiveenglishksa
Version: 1.0.0+1

## Important
This is a Flutter WebView wrapper. It opens the live web app from masarsa.online.

## Codemagic
1. Upload this project to GitHub.
2. Connect repository to Codemagic.
3. Add Apple Developer signing settings.
4. Build iOS IPA.
5. Upload to TestFlight.

## If Flutter iOS folder is incomplete
Run:
flutter create .
flutter pub get
Then replace lib/main.dart with the included main.dart.
